# audition

### Requirements

- CocoaPods 0.39.0
    - For installation see: https://cocoapods.org/
- Xcode 7.3.1
     - Swift 2
- Internet connection

### Running

1. Clone the repo
1. `cd <download location>/audition/Audition`
1. run `pod install`
1. open the workspace file
1. Run the project
